#  Simple JS Minifier

[![License: MIT](https://shields.io)](https://opensource.org)

ブラウザ上で即座に動作する、インストール不要のJavaScript圧縮ツールです。
正規表現を用いて、コードの可読性を保ったままファイルサイズを最小化します。

##  特徴

- **No Install**: Node.jsやnpmは不要。ブラウザさえあればどこでも動きます。
- **Fast Execution**: 正規表現による高速な一括置換処理。
- **Privacy Safe**: 外部サーバーへコードを送信しません。すべての処理はあなたのブラウザ内で完結します。
- **Feature Set**:
    - マルチライン/シングルラインコメントの完全除去。
    - 演算子（`+`, `=`, `{` 等）前後の不要な空白を最適化。
    - 改行および冗長なスペースの削除。

##  クイックスタート

1. このページに開きます。　https://frisk-v3.github.io/js-assyuku/
2. `Input` エリアにJavaScriptを貼り付け、`Minify` ボタンを押してください。



##  仕組み (正規表現)

本ツールは以下のステップで圧縮を行います：
1. `/\/\*[\s\S]*?\*\//g` : ブロックコメントの削除
2. `/(^|[^:])\/\/.*/g` : 行コメントの削除
3. `/\s*([{}()\[\]=+\-*/%,;:<>!&|?])\s*/g` : 記号周りの余白削除

##  注意事項

- 変数名や関数名の置換（Mangle機能）には対応していません。
- テンプレートリテラル内の改行など、一部の特殊な構文で意図しない挙動になる場合があります。
